#  Society Management System (SMS)

A full-stack web application developed to efficiently manage housing society operations such as maintenance, payments, complaints, and reports. This system digitizes society workflows and provides transparency, accuracy, and ease of management for administrators and members.

---

##  Features

###  User & Admin Management
- Secure login and authentication system
- Role-based access (Admin / Member)
- Session management

###  Maintenance Management
- Monthly maintenance entry
- Charge-wise payment tracking
- Remaining balance calculation
- Receipt generation

###  Receipts & Payments
- Payment records storage
- Auto-generated receipts
- Razorpay payment gateway integration *(optional / advanced)*

###  Dashboard & Reports
- Society financial summary dashboard
- Monthly and yearly reports
- Animated counters and progress indicators

###  Complaints Management
- Members can submit complaints
- Admin can view and resolve issues
- Status tracking

###  UI & UX
- Responsive design using Bootstrap 5
- Clean dashboard layout
- Dark/Light mode support *(if enabled)*

##  Architecture

The project follows **MVC (Model–View–Controller) principles**:

- **Model:** MySQL database & PHP database queries
- **View:** HTML, CSS, Bootstrap-based UI
- **Controller:** PHP scripts handling business logic, validation, and routing
